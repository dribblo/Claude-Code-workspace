# Template: Meeting Brief

> Use this format whenever preparing for a meeting.

## Format

```
**Meeting:** [Title / purpose]
**With:** [Attendees]
**Date:** [Date and time]

**Objective:** [What needs to be decided or achieved]

**Key points to make:**
1. [Point]
2. [Point]
3. [Point]

**Questions to ask:**
1. [Question]
2. [Question]

**Anticipated pushback:**
- [Concern] → [Your response]

**Decisions needed:**
- [Decision 1]
- [Decision 2]

**Follow-ups after meeting:**
- [ ] [Action item]
```
