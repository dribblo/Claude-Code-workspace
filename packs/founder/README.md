# Founder Pack

Pre-built modes for startup founders and CEOs.

## Included modes

- **work.md** — Day-to-day founder work, product, strategy
- **investor.md** — Investor updates, fundraising, board prep
- **hiring.md** — Job descriptions, interviews, candidate communication

## How to use

The wizard will detect this pack and offer it as a starting point. You can customise each mode during setup.
